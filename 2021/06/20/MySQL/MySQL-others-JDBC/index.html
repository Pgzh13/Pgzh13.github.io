MySQL-D-其他工具

#MySQL高级

##其他

###JDBC

--说明：Java Database Connectivity ,Java和数据库的连接技术，sun公司推出的一套java应用程序访问数据库的计数规范


--使用步骤：
    a.前提：准备mysql的驱动包，加载到项目中

    b.复制mysql-connector-java-5.1.37-bin.jar到项目的根目录下或libs目录下，然后右击build path-add to build path

    c.加载驱动：
        1-DriverManager.registerDriver(new Driver());（使用new方式加载类）
            <!-- 缺点：
                    （1）属于编译期加载，如果编译期间该类不存在，则直接报编译错误，依赖性太强
                    （2）其会驱动底层静态代码块，其中有一个Driver，会导致Driver对象创建了两遍，效率较低
             -->

        2-Class.forName(驱动包)；（通过反射方式来加载类）
            <!-- 优点：
                    （1）属于运行期加载，大大降低了类的依赖性
                    （2）Driver仅创建1遍。效率较高
             -->

    d.获取连接
        Connection connection = DriverManager.getConnection(......);

    e.执行增删改查
        Statement statement = connection.createStatment();
        int state = statement.executeUpdate(sql);
            <!-- 执行查询操作的时候，通过ResultSet流来获取结果集，获取结果集的值有：
                    （1）通过列的索引
                    （2）通过列名 
            -->
    f.关闭连接
        statement.close();
        connection.close();
